Sobre este Repositório

  • Propósito: Facilitar o acesso de agentes de IA a atos normativos
  • Limitação técnica: Sistemas de ingestão de conhecimento frequentemente limitam URLs a 2 níveis de profundidade
  • Solução: URLs rasas (ex: usuario.github.io/repositorio/documento.html)
  • Autenticidade: Sempre consulte as fontes oficiais do TJPR para validação jurídica

Aviso: Este repositório é um projeto em desenvolvimento e serve apenas para facilitar a consulta de agentes de IA. Para uso jurídico, sempre consulte as fontes oficiais. O desenvolvedor não se responsabiliza por eventuais divergências ou falta de atualizações; o usuário deve verificar as fontes e os espelhos e comparar o conteúdo antes do uso.
