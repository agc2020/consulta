# Consulta ‚Äì Structured Repository of Brazilian Normative Acts

(ENGLISH VERSION FIRST)

The **Consulta** project provides extraction, standardization, structuring and publication of Brazilian normative acts (laws, decrees, resolutions, normative instructions, judicial acts, etc.) in **HTML, JSON and TXT** formats, enriched with metadata and optimized for static search using **Pagefind**.

This repository does **not** change the content of the original legal texts.  
It only **structures, organizes and standardizes** public-domain documents for research, accessibility and integration with AI tools.

## Purpose of the Project

- Extract legal and normative documents from official sources  
- Normalize and structure documents  
- Generate:
  - semantic HTML mirrors  
  - JSON files with hierarchical metadata  
  - TXT compilations  
- Provide a static, efficient search engine using Pagefind  
- Enable use as a structured knowledge base for research and AI models
